Admin Panel 
-------
_**Чтобы начать работу:**_

- Перейди в папку app: 
```
   cd app
```
- Необходимо установить проект:
```
   npm install
```
Для запуска клиента использовать:
```
   npm start
```
---

- Перейди в папку server: 
```
   cd server
```

- Необходимо установить проект:
```
   npm install
```

Для запуска сервера использовать:
```
   npm run dev
```
---
