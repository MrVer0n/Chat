Admin Panel 
-------
_**Чтобы начать работу:**_

- Перейди в папку app: 
```
    cd app
```
- Необходимо установить проект:
```
   npm install
```
---
Для запуска использовать:
```
    npm start
```

Для остановки в консоле сочетание клавиш:
```
    Ctrl+C
```

Для запуска сервер(в разработке):
```
    npm run dev
```
---
